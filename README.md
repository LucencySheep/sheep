## 自己写的JFXDialog
<br>由于原本的JFXDialog创建起来有点麻烦，所以我自己封装了一个
<br>只需要 **new MyDialog((父窗口)Stage stage，(按钮信息)String，(提示信息)String)**  就可以创建一个JFXDialog出来了
<br>运行本程序需要先导入好两个jar包:  **jfoenix-8.0.8.jar , jfxrt.jar**        
<br>建议用法：放在方法里new,然后调用这个方法
